Observables (RxJS)

  • Stream of
    • any number of things
    • over any length of time
  • Helps With
    • Memory Leaks (unsubscribe)
    • Race Conditions
    • Complex State Machines
    • Error Handling
  • Observables declare intent
  • Lazy (requires Subscription)
  • Fully cancellable (including tear-down)
  • Orchestration (like LoDash for async)
